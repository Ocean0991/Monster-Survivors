Monster Survivors

Dive into an epic underwater survival adventure where mythical sea creatures rule the depths. Can you survive the ocean's deadly challenges?

About Monster Survivors

Monster Survivors is an immersive underwater survival game that challenges players to navigate treacherous ocean depths while evading and battling colossal sea creatures. Combining elements of strategy and action RPG, the game features:

  • Dynamic ecosystem with intelligent AI predators
  • Underwater base-building mechanics
  • Resource management and crafting system
  • Multiplayer survival challenges

Gameplay Highlights

Experience real-time underwater combat with adaptive difficulty. The game's procedural generation system creates unique challenges every playthrough, ensuring endless replayability.

System Requirements

  1. Windows 10 64-bit or macOS Monterey
  2. 4GB RAM minimum (8GB recommended)
  3. WebGL 2.0 compatible browser
  4. Stable internet connection
